# CMG Wheelchair Official Website

This project is a high-fidelity landing page for the CMG Wheelchair, inspired by the premium aesthetic and user-focused design of WHILL.inc.

## Structure
- `index.html`: Main landing page
- `css/style.css`: Custom styles (Tailwind CSS via CDN for rapid prototyping)
- `js/main.js`: Interactivity and animations
- `assets/`: Images and icons

## Core Tech
- Stability Control via Control Moment Gyro (CMG)
- Omni-directional mobility
- Intelligent obstacle avoidance
